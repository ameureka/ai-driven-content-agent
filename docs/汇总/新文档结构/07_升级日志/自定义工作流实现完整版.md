## ✅ 已完成的功能

### 1. 后端增强
- **环境变量扩展**: 支持 `CUSTOM_WORKFLOWS` JSON 配置自定义工作流
- **WorkflowManager 类**: 统一管理默认和自定义工作流配置
- **动态 API 路由**: `/api/v1/workflows/{workflowId}/execute` 支持任意工作流 ID
- **新增 API 端点**:
  - `GET /api/v1/workflows/available` - 获取可用工作流列表
  - `POST /api/v1/workflows/custom` - 添加自定义工作流
  - `DELETE /api/v1/workflows/custom/{workflowId}` - 删除自定义工作流

### 2. 前端工作流选择器

### 3. 智能路由逻辑

- **类型安全**: 严格的工作流类型验证和错误提示
- **流式/阻塞双模式**: 保持现有的流式处理能力
