# AI驱动内容代理 - 部署与运维完整指南

## 🚀 项目概述

**AI驱动内容代理**是一个基于Cloudflare Workers的智能内容处理服务，集成了Dify AI平台，提供Markdown到HTML的智能转换和多样化模板渲染功能。该项目采用现代化的无服务器架构，具备高性能、高可用性和易扩展的特点。

### 核心特性

- **智能内容转换**: 支持Markdown到HTML的智能转换
- **AI驱动**: 集成Dify AI平台，提供智能内容生成和处理
- **多模板支持**: 提供多种渲染模板，适应不同场景需求
- **无服务器架构**: 基于Cloudflare Workers，零运维成本
- **高性能存储**: 使用KV存储和R2对象存储，确保快速响应
- **现代化前端**: 响应式设计，支持多设备访问

### 技术栈

- **运行时**: Cloudflare Workers (V8 JavaScript Runtime)
- **存储**: Cloudflare KV + R2 Object Storage
- **AI平台**: Dify AI API
- **前端**: HTML5 + CSS3 + Vanilla JavaScript
- **构建工具**: Wrangler CLI + ESBuild
- **测试框架**: Vitest

### 项目架构

```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   用户请求      │───▶│  Cloudflare     │───▶│   Worker        │
│   (Browser)     │    │   Edge Network  │    │   Runtime       │
└─────────────────┘    └─────────────────┘    └─────────────────┘
                                                        │
                                                        ▼
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   R2 Storage    │◀───│   KV Storage    │◀───│   AI Service    │
│  (Static Files) │    │  (Cache/Data)   │    │   (Dify API)    │
└─────────────────┘    └─────────────────┘    └─────────────────┘
```

**数据流说明**:
1. 用户通过浏览器发送请求到 Cloudflare Edge Network
2. Edge Network 将请求路由到最近的 Worker 实例
3. Worker 处理请求，可能调用 Dify AI API 进行内容生成
4. 使用 KV 存储进行数据缓存和状态管理
5. 从 R2 存储获取静态资源（HTML、CSS、JS）
6. 返回处理结果给用户

## 📁 配置文件说明

### 核心配置文件

#### 1. `wrangler.toml` - Cloudflare Workers 主配置

**作用**: Cloudflare Workers 的核心配置文件，定义了 Worker 的部署参数、资源绑定和环境设置。

**主要配置项**:
```toml
name = "ai-driven-content-agent"       # Worker 名称
main = "src/index.js"                  # 入口文件
compatibility_date = "2025-08-06"      # 兼容性日期
compatibility_flags = ["nodejs_compat"] # Node.js 兼容性

# KV 存储绑定
[[kv_namespaces]]
binding = "MARKDOWN_KV"                         # 代码中的绑定名
id = "e8212bb5681840a3a44e1bfff52fd505"         # 生产环境 KV ID
preview_id = "ed67abba3b504765839b8c7391fe4e5c" # 预览环境 KV ID

# R2 存储绑定
[[r2_buckets]]
binding = "ASSETS"                     # 静态资源绑定
bucket_name = "ai-driven-content-agent-assets" # 存储桶名称
```

## 🔒 安全配置

### API 安全

#### 请求验证

```javascript
// API 密钥验证
const validateApiKey = (request) => {
  const apiKey = request.headers.get('Authorization')?.replace('Bearer ', '');
  if (!apiKey || apiKey !== API_KEY) {
    throw new Error('Unauthorized');
  }
};

// 请求频率限制
const rateLimiter = {
  async checkLimit(clientId) {
    const key = `rate_limit:${clientId}`;
    const current = await MARKDOWN_KV.get(key);
    const count = current ? parseInt(current) : 0;
    
    if (count >= 100) { // 每分钟100次请求限制
      throw new Error('Rate limit exceeded');
    }
    
    await MARKDOWN_KV.put(key, (count + 1).toString(), { expirationTtl: 60 });
  }
};
```

```javascript
// 安全的 CORS 配置
const corsHeaders = {
```

```javascript
// 数据加密存储
const encryptData = async (data) => {
  const encoder = new TextEncoder();
  const key = await crypto.subtle.importKey(
    'raw',
    encoder.encode(ENCRYPTION_KEY),
    { name: 'AES-GCM' },
    false,
    ['encrypt']
  );
  
  const iv = crypto.getRandomValues(new Uint8Array(12));
  const encrypted = await crypto.subtle.encrypt(
    { name: 'AES-GCM', iv },
    key,
    encoder.encode(data)
  );
  
  return { encrypted, iv };
};
```
